Dados Enviados pelo Webhook
Conforme solicitado, aqui está a estrutura dos dados que são enviados via POST para a sua URL de webhook sempre que uma nova análise da IA é concluída. Você pode usar essa estrutura para integrar os insights da IFINANCE com outros sistemas, como Slack, CRMs ou bancos de dados.

Endpoint: https://your-webhook-endpoint.com/api/financial-analysis (configurado no código)
Método: POST
Corpo da Requisição (JSON):
